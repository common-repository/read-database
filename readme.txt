Read database give capability to WordPress dashboard to read data inside tables of WordPress.

== Description ==
Read Database is very simple and low weight plugin which helps WordPress/dashboard user to check data inside tables, user can make settings to exclude/hide tables if they want not to read them. Also data can be downloaded in CSV format.

Features:

*  	Read data of WordPress tables.
*  	Set which tables needs to read data asynchronously.
*  	Download table data in CSV format.

== Installation ==
*	Download the plugin .zip file.
*	Login in to admin Click Plugins -> Add New -> Upload.
*	Find Read Database in plugin list  and activate it.
*	After activate plugin you will see Read Database menu at left side on WordPress dashboard.
